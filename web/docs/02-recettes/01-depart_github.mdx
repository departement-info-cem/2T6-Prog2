## Crée un compte GitHub

1. Allez sur [GitHub](https://github.com/)
2. Entrez votre adresse courriel du cégep ( matricule@cegepmontpetit.ca ) dans la barre "Sign up for GitHub" et appuyez sur Entrez
    ![Barre Sign up](./images/github_signup.png)
3. Ouvrez vos courriels et récupérez le code de vérification.
4. Entrez le code de vérification dans le formulaire d'inscription.

![Compte étudiant GitHub](./images/github_studentPack.png)

5. (Optionnel pour ce cours) Activez le forfait étudiant.
    - Allez sur [GitHub Education](https://education.github.com/pack/join)
    - Cliquez sur "Yes, I'm a student"
    - GitHub va vous demandez de fournir une preuve, vous pouvez utiliser une photo de votre carte étudiante.



## Télécharger GitHub Desktop

Pour travailler à la maison, vous aurez besoin d'utiliser un logiciel pouvant interagir avec GitHub.

Dans le cadre du cours nous utiliserons GitHub Desktop. Vous *pouvez* passer par VSCode ou utiliser un autre logiciel, mais il n'y aura pas de support si vous rencontrez des problèmes.

1. Allez Télécharger l'installateur de [GitHub Desktop](https://desktop.github.com/download/).
2. Exécutez GitHubDesktopSetup-x64.exe et suivez les instructions de l'installateur.
